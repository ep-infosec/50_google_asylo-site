enum asylo::sgx::AttributeBit
strong

SGX defines 128 bits of enclave attributes, which are located in the SECS (Secure Enclave Control Structure) of the enclave.

The lower 64 bits of these attributes are treated as individual flags, whereas the upper 64 bits are collectively called XFRM (XSAVE Feature Request Mask). This enum defines the various attribute bits and assigns them a value that is same as their bit position in the SECS attributes bit vector. The names of these bits are taken verbatim from the Intel SDM (Software Developer's Manual), volume 3D (see https://software.intel.com/en-us/articles/intel-sdm).

Enumerator
INIT 

Indicates whether the enclave has been initialized via EINIT instruction.

DEBUG 

Indicates whether the enclave is a debug (1) or production (0) enclave.

MODE64BIT 

Indicates whether the enclave is a 64-bit (1) or a 32-bit (0) enclave.

PROVISIONKEY 

Indicates whether the enclave has access to the SGX provisioning key (1) or not (0).

INITTOKENKEY 

Indicates whether the enclave has access to the INIT-token key (1) or not (0).

KSS 

Indicates whether the enclave has support for Key Separation and Sharing (KSS) (1) or not (0).

Enabling KSS sets the ISVEXTPRODID, ISVFAMILYID, CONFIGID and CONFIGSVN values in an enclave's identity.

enum asylo::sgx::MiscselectBit
strong

The following enum defines the various MISCSELECT bits and assigns them a value that is same as their bit position in the SECS MISCSELECT bit vector.

The names of these bits are taken verbatim from the Intel SDM (Software Developer's Manual).

Enumerator
EXINFO 

Indicates that information about page faults and GP exceptions that occurred inside an enclave will be saved upon an asynchronous exit.
