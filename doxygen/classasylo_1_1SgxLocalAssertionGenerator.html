Detailed Description

An implementation of the EnclaveAssertionGenerator interface for SGX local assertions.

An SgxLocalAssertionGenerator is capable of generating assertion offers and assertions for SGX code identities that can be verified by SGX enclaves running within the same local attestation domain.
